# Data Platform UI (React + Tailwind)

Matches the screenshot structure:
- Dark top navbar
- **Data** mega menu (3 columns)
- Light content pages with left sidebar navigation
- Pages: Market Data (Equities & ETFs, Futures + placeholders), Reference Data, Data Analytics, Coverage, Developers

## Run
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```
